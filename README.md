# Shree Sai Creation

Premium, data-driven frontend for a luxury chandelier and decorative lighting brand.

## Run locally

```bash
npm install
npm run dev
```

The typed content layer is in `data/site-data.ts`; replace those exports with CMS/API fetchers when the backend is ready. Contact details are centralized in the `settings` object.

## Routes

`/`, `/about`, `/collections`, `/collections/[slug]`, `/projects`, `/bespoke`, `/blog`, `/blog/[slug]`, `/contact`, and a custom 404 state.
